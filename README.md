# 📊 Simple Finance Tracker with Python

Proyek ini adalah langkah awal saya dalam perjalanan menjadi seorang **Data Analyst**. Di sini, saya menggunakan Python untuk mengotomatisasi perhitungan keuangan pribadi.

### 🚀 Fitur
- Menghitung sisa saldo dari pemasukan dan pengeluaran.
- Menghitung persentase tabungan terhadap total pendapatan.
- Memberikan analisis sederhana berdasarkan rasio tabungan.

### 🛠️ Teknologi yang Digunakan
- **Bahasa:** Python 3.13
- **Tools:** Visual Studio Code

### 📈 Hasil Eksekusi
Berikut adalah contoh output dari program:
- Total Pemasukan: Rp5,000,000
- Total Tabungan: Rp3,000,000
- Rasio Hemat: 60%
